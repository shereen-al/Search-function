How to Use

Simply start typing in the search box above. As you type, matching text will be highlighted throughout the page. Use the navigation arrows to jump between different matches, or press Enter to cycle through results.

The search function scans all visible text content including headings, paragraphs, list items, and other text elements. It's designed to be fast and responsive, providing immediate feedback as you search.
